• Linguagem de Programação

HTML

HTML (HyperText Markup Language)
É uma linguagem utilizada para criar páginas web.
Ele consiste em um conjunto de marcações (tags) que definem a estrutura e o conteúdo da página. As tags são interpretadas pelos navegadores para exibir o conteúdo adequadamente.

CSS

(Cascading Style Sheets)
É uma linguagem de estilo utilizada para controlar a apresentação e o design de páginas web.
Ele trabalha em conjunto com o HTML, permitindo definir cores, fontes, layout, espaçamento e outros aspectos visuais de elementos HTML.

JAVA SCRIPT

JavaScript é uma linguagem de programação de alto nível, orientada a objetos e amplamente utilizada para desenvolvimento web.
Ela permite adicionar interatividade, comportamento dinâmico e funcionalidades avançadas a páginas web.

JAVA

Java é uma linguagem de programação versátil, portátil e orientada a objetos.
Java é amplamente utilizado no desenvolvimento de aplicativos de desktop, sistemas embarcados, aplicações web e desenvolvimento móvel. Ela possui uma biblioteca padrão abrangente uma variedade de bibliotecas e frameworks adicionais.
