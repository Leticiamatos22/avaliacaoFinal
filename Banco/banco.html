    O que é Banco de Dados

    Banco de dados é uma coleção organizada de informações
    estruturadas que são armazenadas e acessadas eletronicamente.
    Ele é projetado para armazenar, gerenciar e recuperar
    dados de maneira eficiente e segura.
    Um banco de dados é composto por tabelas, que são
    estruturas que contêm registros (linhas) e campos (colunas).
