    O que é Banco de Dados

      Banco de dados é uma coleção organizada de informações estruturadas que são armazenadas e acessadas eletronicamente. Ele é projetado para armazenar, gerenciar e recuperar dados de maneira eficiente e segura. Um banco de dados é composto por tabelas, que são estruturas que contêm registros (linhas) e campos (colunas).

    IDEs Utilizadas

      Durante esse semestre para o aprendizado da criação do banco utilizamos o PostgreSQL, que é um sistema de gerenciamento de banco de dados relacional (SGBDR). O PostgreSQL suporta a linguagem SQL (Structured Query Language) e ele é usado em uma variedade de aplicações. Para o gerenciamento do mesmo, utilizamos a IDE pgAdim4, que é uma ferramenta de administração e desenvolvimento de banco de dados para o PostgreSQL. Ele fornece uma interface gráfica amigável para gerenciar e manipular bancos de dados.
