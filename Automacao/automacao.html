    O que é Automação Industrial

    Automação industrial refere-se ao uso de tecnologia e sistemas
    para controlar e operar processos industriais de forma automatizada.
    É o emprego de máquinas, computadores e softwares para executar tarefas
    que antes eram realizadas manualmente por operadores humanos.

    IDEs Utilizadas


    Durante esse semestre para o aprendizado da criação do banco utilizamos
    o PostgreSQL, que é um sistema de gerenciamento de banco de dados relacional (SGBDR).
    O PostgreSQL suporta a linguagem SQL (Structured Query Language) e ele
    é usado em uma variedade de aplicações.
    Para o gerenciamento do mesmo, utilizamos a IDE pgAdim4, que é uma ferramenta
    de administração e desenvolvimento de banco de dados para o PostgreSQL.
    Ele fornece uma interface gráfica amigável para gerenciar e manipular bancos de dados.
